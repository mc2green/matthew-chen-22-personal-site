Skills in Progress: C++ & jQuery

HTML5

Hyper Text Markup Language is used to modify the structure, content, and style of a web page by using elements and tags to create the Document Model Object (DOM).

CSS3

Cascading Style Sheets customize how the HTML elements are displayed on the DOM and provide for an inviting and engaging user interface.

Javascript

Javascript is a scripting language that creates dynamic web pages by updating HTML and CSS on the client-side and accessing APIs on the server-side.

PostgreSQL/Sequelize.js

PostgreSQL is an object-relational management system that extends SQL to store and scale data. Sequelize is a promised-based ORM used to access numerous database types using Javascript.

Node.js/Express.js

Node.js is a powerful runtime enviornment that introduces Javascript to the server-side. Express.js is a lightweight web applications framework for building RESTful APIs

React & React Native

React is a Javascript library that uses JSX to render HTML-like components on the Virtual DOM, thereby improving performance. React Native uses React architecture to design native mobile apps.

Redux

Redux is a centralized state management system that acts as the single source of truth within a large application rendering many React components.

Semanitc UI

Semantic UI is a lightweight, modern front-end development framework that allows for peak customization of themes and elements with high readability.

Bootstrap4

Bootstrap4 is a popular CSS front-end framework with a vast array of design templates for HTML elements in order to create beautiful and responsive interfaces.

Google Maps API

Google Maps API is a collection of APIs behind the features of Google Maps and any application can use its extensive geographical data.

Heroku

Heroku is a convenient cloud platform that can launch and host both front-end and back-end aspects of an application for free.

OAuth

OAuth is a open-standard authorization protocol that enables secure, third-party authorization to authenticate a user while logging in.
